TP Windows Server - Active Directory

Contexte

Date : 05/05/2025 au 11/05/2025

Cadre : École - TP systèmes

Mise en place d'un environnement Active Directory.

Objectif

Mettre en place un environnement Active Directory pour l'entreprise AssurTech (4 sites) avec gestion centralisée des utilisateurs, groupes, droits et déploiement de logiciels.

Mon rôle

Administratrice système - Travail individuel sur VirtualBox (Windows Server 2019 + Windows 10).

Réalisations

  • Installation et configuration Windows Server 2019 (IP statique, nom machine)
  • Déploiement rôles ADDS, DNS, DHCP
  • Promotion en contrôleur de domaine (assurtech.local)
  • Configuration DNS et DHCP
  • Création structure AD : 4 OUs (TLS, MRS, PAR, BDX), 7 groupes, 12 utilisateurs
  • Configuration dossiers partagés avec permissions par groupe
  • Déploiement logiciels via GPO (Teams, 7-Zip)
  • Application GPO de sécurité (désactivation CMD sauf service Technique)
  • Jonction client Windows 10 au domaine et tests

Résultats

Environnement Active Directory fonctionnel avec 12 utilisateurs répartis dans 7 groupes sur 4 sites. Services DNS et DHCP opérationnels. Déploiement automatique de logiciels et restrictions de sécurité appliquées. Tests validés sur poste client.

Technologies

Windows Server 2019 Active Directory DNS DHCP GPO PowerShell

Compétences BTS SIO mobilisées

✓ Gérer le patrimoine informatique

Gestion centralisée utilisateurs/groupes, configuration services réseau, documentation.

✓ Répondre aux incidents et aux demandes d'assistance

Configuration environnement structuré facilitant le support, gestion permissions, tests.

✓ Mettre à disposition des utilisateurs un service informatique

Déploiement services opérationnels (DHCP, DNS), dossiers partagés, déploiement logiciels.
