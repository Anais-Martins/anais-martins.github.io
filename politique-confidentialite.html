Politique de confidentialité

1. Introduction

La présente politique de confidentialité décrit la manière dont ce site personnel — portfolio d'Anaïs Martins, hébergé sur GitHub Pages — traite les données des visiteurs. Elle a été rédigée dans le respect du Règlement Général sur la Protection des Données (RGPD) en vigueur depuis le 25 mai 2018.

2. Données collectées par ce site

Ce site ne collecte aucune donnée personnelle des visiteurs. Il n'existe ni formulaire de contact, ni système d'inscription, ni compte utilisateur, ni outil de suivi analytique (Google Analytics ou équivalent).

Aucun cookie n'est déposé sur votre navigateur par ce site.

3. Hébergement – GitHub Pages

Ce site est hébergé par GitHub Pages (GitHub, Inc. — 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, États-Unis). En tant qu'hébergeur, GitHub est susceptible de collecter certaines données techniques liées à la navigation (adresse IP, navigateur, pages consultées) à des fins de sécurité et de gestion de l'infrastructure, indépendamment de la volonté de l'éditrice de ce site.

Pour en savoir plus sur la politique de confidentialité de GitHub : docs.github.com – Déclaration de confidentialité GitHub

4. Vos droits

Conformément au RGPD, vous disposez d'un droit d'accès, de rectification et de suppression de vos données personnelles. Ce site ne collectant aucune donnée, ces droits n'ont pas vocation à s'exercer ici. Pour toute question, vous pouvez néanmoins contacter l'éditrice :

Email : anais.martins35@gmail.com
