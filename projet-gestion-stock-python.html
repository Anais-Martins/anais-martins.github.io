Gestionnaire de stock de marchandises

Contexte

Date : 08/04/2025 au 02/05/2025

Cadre : École - Projet personnel - Cours d'algorithmique

Développement d'une application de gestion de stock.

Objectif

Développer une application Python de gestion de stock pour une épicerie avec système d'alertes et calcul de valeur.

Mon rôle

Développeuse - Travail individuel.

Réalisations

  • Création d'un inventaire avec dictionnaires imbriqués (produit, quantité, prix)
  • Développement de 5 fonctions : consultation inventaire, entrée/sortie stock, alertes, calcul de valeur
  • Gestion d'erreurs utilisateur avec boucles de validation
  • Système d'alertes avec seuils personnalisés par produit
  • Interface menu interactif en ligne de commande

Résultats

Application fonctionnelle testée avec 4 produits. Code respectant les bonnes pratiques (commenté, indenté). Utilisation de types variés (int, float, str, bool) et structures de données (dictionnaires, listes).

Technologies

Python Module datetime Dictionnaires

Compétences BTS SIO mobilisées
