Situation professionnelle #1 – Épreuve E6

Contexte

Date : 01/12/2025 au 04/04/2026

Cadre : École – IRIS Mediaschool Toulouse – Épreuve E6 du BTS SIO SLAM

Organisation support : École du cinéma de Toulouse (contexte fictif)

Modalité : Travail individuel

L'École du cinéma de Toulouse souhaitait moderniser sa gestion administrative en développant une application web permettant de centraliser et optimiser la gestion de ses formations (Acting, Réalisation, Scénarisation).

Objectif

Développer une application web de gestion de formations en architecture client léger (MVC), avec une base de données MySQL relationnelle, une API REST complète et une interface utilisateur responsive.

Mon rôle

Développeuse fullstack – Travail individuel. Conception et développement complets de l'application, de la base de données à l'interface utilisateur.

Réalisations

  • Conception d'une base de données relationnelle MySQL de 14 tables (apprenants, formateurs, formations, cours, salles, inscriptions, présences, avis, notes, utilisateurs, tables de liaison)
  • Création des modèles Sequelize avec gestion des associations (hasMany, belongsTo)
  • Développement des routes API REST avec implémentation CRUD complète pour toutes les entités
  • Développement des routes EJS frontend (liste, ajouter, modifier) pour chaque entité
  • Création des vues EJS dynamiques avec Bootstrap 5 pour le design responsive
  • Mise en place d'un système d'authentification par rôle (Admin, Formateur, Apprenant)
  • Sécurisation de l'application avec HTTPS (certificat SSL, cert.pem / key.pem)
  • Tests des routes API avec Postman
  • Espace administrateur avec gestion des formateurs, apprenants, inscriptions, cours et utilisateurs

Résultats

  • Application web fonctionnelle et déployable : accessible en HTTPS sur localhost:3001, prête à être présentée à l'épreuve E6
  • Base de données relationnelle cohérente : 14 tables avec associations complexes gérées via Sequelize
  • API REST complète : opérations CRUD pour toutes les entités, testées et validées via Postman
  • Interface utilisateur responsive : espace administrateur complet permettant la gestion de toutes les entités depuis le navigateur
  • Authentification par rôle opérationnelle : accès différenciés pour Admin, Formateur et Apprenant

Technologies

Node.js Express.js Sequelize (ORM) MySQL EJS Bootstrap 5 JavaScript API REST Postman phpMyAdmin XAMPP HTTPS / SSL
