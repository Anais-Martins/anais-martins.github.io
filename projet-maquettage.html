Maquettage

Contexte

Date : 01/04/2026 au 02/04/2026

Cadre : Entreprise – CIAG

CIAG est une ESN qui intègre le logiciel de gestion Divalto (ERP/CRM) chez ses clients. Dans le cadre d'une demande client, j'interviens sur la pièce comptable commande pour modifier sa maquette d'impression via le SDK Harmony.

Le client souhaite deux modifications : remplacer le champ INCOTERMS (situé sous le cartouche de coordonnées du destinataire) par le Code affaire, et que ce champ soit automatiquement traduit lors de la génération d'une commande pour un client étranger, en fonction de la langue paramétrée dans sa fiche client.

Objectif

Modifier la maquette de la pièce commande dans le SDK Divalto pour remplacer le champ INCOTERMS par le Code affaire, et implémenter la traduction automatique de ce libellé selon la langue du client destinataire.

Mon rôle

Développeuse / Intégratrice Divalto – Travail individuel.

Réalisations

  • Génération d'une commande dans Divalto pour identifier, à partir de l'impression, le fichier maquette concerné
  • Création d'un nouveau champ dans la base de données : ci_lib10 (12 caractères) dans la table ci_trav, destiné à accueillir le libellé du Code affaire
  • Ajout de l'instance du nouveau champ sur la maquette dans le SDK Harmony
  • Ajout de la traduction du libellé dans le traitement de la maquette (code), pour chacune des langues concernées
  • Création de commandes de test dans Divalto avec des clients paramétrés en différentes langues
  • Tests d'impression sur les différentes commandes générées et vérification du rendu

Résultats

  • Champ INCOTERMS remplacé par le Code affaire : la maquette affiche désormais le bon champ à l'emplacement souhaité par le client
  • Traduction automatique fonctionnelle : le libellé s'adapte à la langue paramétrée dans la fiche client lors de la génération de la commande
  • Tests concluants : vérification sur plusieurs commandes avec des clients de langues différentes, résultats conformes dans chaque cas
  • Livrable conforme au cahier des charges : la maquette est validée et prête à être déployée chez le client

Technologies
