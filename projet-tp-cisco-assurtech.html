TP Infrastructure réseau - AssurTech

Contexte

Date : 05/05/2025 au 11/05/2025

Cadre : École - TP réseaux

Configuration de l'infrastructure réseau d'une société d'assurance.

Objectif

Concevoir et configurer l'infrastructure réseau complète d'une société d'assurance fictive sur 4 sites (Toulouse, Bordeaux, Marseille, Paris) avec segmentation par services et communication inter-sites sécurisée.

Mon rôle

Administratrice réseau - Travail individuel sur Cisco Packet Tracer.

Réalisations

  • Configuration de 5 routeurs Cisco (1 par site + 1 central)
  • Création de 11 VLANs (segmentation par service + WiFi invités)
  • Plan d'adressage IP structuré
  • Configuration routage statique inter-sites sur routeur central et sites
  • Déploiement services DHCP, DNS, Web (HTTP)
  • Sécurisation accès SSH sur tous les routeurs
  • Configuration WiFi interne (par service) et WiFi invités (par site)
  • Tests de connectivité et validation des règles de communication

Résultats

Infrastructure opérationnelle avec communication inter-sites fonctionnelle. Service Technique (Marseille) accessible depuis tous les sites. Communication bidirectionnelle Toulouse ↔ Paris. Documentation technique complète avec schéma réseau, plan d'adressage et configurations.

Technologies

Cisco Packet Tracer Routeurs Cisco VLANs Routage statique DHCP DNS SSH
