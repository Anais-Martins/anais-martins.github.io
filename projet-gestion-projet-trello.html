Gestion de projet - Refonte site école

Contexte

Date : 12/11/2025 au 13/11/2025

Cadre : École - Cas pratique - Cours de gestion de projet

Planification d'un projet de refonte du site web d'une école.

Objectif

Planifier et organiser un projet de refonte complète du site web d'une école en appliquant les méthodologies de gestion de projet (contraintes : délai 3 mois, budget 15 000€, équipe 4 personnes).

Mon rôle

Manager de projet / Design - Travail en équipe de 4 personnes. Responsabilités : gestion globale du projet et création graphique.

Réalisations

  • Choix méthodologie Agile (adaptée au web, validation étape par étape, délai court)
  • Définition de 5 fonctionnalités prioritaires : page d'accueil, formations, stage & alternance, à propos, contact
  • Découpage en 15 tâches réparties sur 3 phases (préparation 2 sem, développement 6 sem, tests 4 sem)
  • Identification et gestion de 3 risques majeurs (départ membre, manque compétences responsive, période fêtes)
  • Répartition des rôles : manager projet/design, développeur front/back, cybersécurité/tests, contenu/SEO
  • Mise en place Trello pour suivi du projet
  • Création dépôt GitHub avec README et index.html

Résultats

Planning complet sur 3 mois avec méthodologie Agile justifiée. Tableau Trello organisé avec tâches réparties. Dépôt GitHub configuré avec tous les membres collaborateurs. Stratégies de gestion des risques définies (réduction via marges, formation, rallongement délai).
